# Java_fundamentals_week2
Java OOP assignment demonstrating core Object-Oriented Programming concepts including classes, inheritance, abstraction, encapsulation, and polymorphism. Contains Student class implementation, Vehicle inheritance example, and a mini Library Management System project using Java.

This repository contains Java programs based on Object-Oriented Programming (OOP) concepts.

## Assignments Included

### 1. Student Class Implementation
- Created a `Student` class
- Used attributes and methods
- Displayed student information

### 2. Vehicle Inheritance Example
- Implemented inheritance using:
  - Base class: `Vehicle`
  - Subclasses: `Car`, `Bike`
- Demonstrated method overriding

### 3. Mini Project – Library Management System
Implemented a simple Library Management System using OOP concepts.

#### Classes Used
- `Book`
- `Library`
- `User`
- `StudentUser`

#### Features
- Add Book
- Borrow Book
- Return Book
- Display Books

## OOP Concepts Demonstrated

### Encapsulation
- Used private variables with getters/setters

### Inheritance
- `StudentUser` extends `User`

### Abstraction
- Abstract class `User`

### Polymorphism
- Method overriding in subclasses
